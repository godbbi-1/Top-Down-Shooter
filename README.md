# Top-Down Shooter

# 프로젝트 개요
Unity와 node.js를 이용한 멀티플레이 탑다운 슈팅게임.

# 구현목표

1. 싱글플레이
2. 멀티플레이(협동)
3. 랭킹

# 진행 과정
서버 연결 완료.
회원가입 완료.
